## ğŸš€ Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/codersclub-scet/coders-club-website.git
cd coders-club-website
```

### 2. Install Dependencies

Using npm:

```bash

```

Or with pnpm:

